# Курсовой проект  
## по дисциплине "Основы программирования и алгоритмизации"  
## по теме «Разработка программы работы с файловой базой данных «IT-компании»»  
Консольная программа на языке C для работы с базой данной, содержащей ключевую информацию об IT-компаниях. Программа предоставляет возможности добавления новых записей через ввод в консоли, генерацию тестовых данных,  
сохранять данные в файл и загружать данные из файла, а также производить сортировку и поиск данных

## Функционал 
### Ввод-вывод и хранение записей
1. Просмотр всех компаний – Форматированный вывод всех записей с детальной информацией
2. Добавление новых компаний – Пошаговый ввод с валидацией через консольный интерфейс
3. Динамическое управление памятью – Автоматическое расширение массива при необходимости
4. Интерактивное меню – Удобная навигация по функциям программы

### Поиск записей
1. Поиск по названию – Точное совпадение наименования компании
2. Комбинированный поиск – Фильтрация по капитализации И рейтингу работодателя одновременно
3. Человекочитаемый вывод – Результаты поиска отображаются в структурированном виде

### Сортировка записей
1. Многоуровневая сортировка по приоритету:
   + Рыночная капитализация (по убыванию – крупнейшие первыми)
   + Рейтинг работодателя (по убыванию – лучшие первыми)
   + Количество сотрудников (по убыванию – крупнейшие первыми)
Быстрая сортировка – Использование оптимизированного алгоритма qsort

### Работа с файлами
1. Сохранение в текстовый файл – Структурированный формат с пояснениями полей
2. Загрузка из файла – Восстановление всей базы данных с автоматическим парсингом
Контроль целостности данных – Проверка корректности загружаемых записей

### Тестирование и отладка
1. Генерация тестовых данных – Автоматическое создание компаний со случайными параметрами
2. Динамическое расширение – Увеличение размера массива при добавлении тестовых записей
3. Защита от переполнения – Контроль границ массива

## Реализованные функции

`const char* sphere_string(Sphere sphere)`
+ Назначение: Преобразование числового значения сферы деятельности в строковое представление
+ Возвращает: указатель на строку с названием сферы

`const char* rate_string(Rate rate)`
+ Назначение: Преобразование числового значения рейтинга в текстовый формат
+ Возвращает: указатель на строку с описанием рейтинга

`void input_company(IT* company)`
+ Назначение: Интерактивный ввод данных о компании с клавиатуры
+ Параметры: company - указатель на структуру для заполнения

`void print_company(const IT* company)`
+ Назначение: Форматированный вывод информации об одной компании
+ Параметры: company - указатель на структуру для отображения

`void print_all_companies(const IT companies[], int count)`
+ Назначение: Отображение всех записей в базе данных
+ Параметры: companies - массив структур, count - количество записей

`int save_companies(const IT companies[], int count, const char* filename)`
+ Назначение: Сохранение всех записей в текстовый файл
+ Параметры: `companies` - массив структур, `count` - количество записей, `filename` - имя файла
+ Возвращает: 0 при успехе, -1 при ошибке

`int load_companies(IT** companies, int* count, int* size, const char* filename)`
+ Назначение: Загрузка записей из текстового файла с автоматическим расширением массива
+ Параметры: `companies` - указатель на массив структур, `count` - текущее количество, `size` - размер массива, `filename` - имя файла
+ Возвращает: 0 при успехе, -1 при ошибке

`int test(IT** companies, int* count, int* size, int n)`
+ Назначение: Генерация тестовых данных и добавление их в базу
+ Параметры: `companies` - указатель на массив, `count` - текущее количество, `size` - размер массива, `n` - количество тестовых записей
+ Возвращает: 0 при успехе, -1 при ошибке выделения памяти

`int comparator(const void* a, const void* b)`
+ Назначение: Функция сравнения для сортировки записей по приоритетам
+ Параметры: `a`, `b` - указатели на сравниваемые структуры
+ Возвращает: -1, 0, 1 в зависимости от результата сравнения

`int search_by_name(const IT companies[], int count, const char* searchname)`
+ Назначение: Поиск компании по точному названию
+ Параметры: `companies` - массив структур, `count` - количество записей, `searchname` - искомое название
+ Возвращает: количество найденных компаний

`int search_by_capandrate(const IT companies[], int count, double min_capitalization, int rate_choice)`
+ Назначение: Комбинированный поиск по минимальной капитализации и рейтингу
+ Параметры: `companies` - массив структур, `count` - количество записей, `min_capitalization` - минимальная капитализация, `rate_choice` - рейтинг для поиска
+ Возвращает: количество найденных компаний

## Особенности реализации
1. Использование перечислений (enum) для типизации сфер деятельности и рейтингов
2. Динамическое выделение памяти с помощью malloc/realloc
3. Обработка ошибок ввода/вывода
